# Құран жаттау

Құран жаттауға арналған қосымша (PWA) — қазақ тілінде, тёмный/светлый режим, сүре бойынша
жаттау, сөздік (278 сөз) іздеу және жаттау режимі, дауыспен тексеру, хатим-жоспар.

Приложение для заучивания Корана. Работает офлайн, устанавливается как приложение (PWA),
из него можно собрать APK для Android.

## Демо / Установка

Открыть в браузере: **https://ВАШ-ЛОГИН.github.io/quran-zhattau/**
(адрес появится после шага «Публикация» ниже)

На Android: открыть ссылку в Chrome → меню (⋮) → **«Установить приложение»**.

## Публикация на GitHub Pages (бесплатно, ~2 минуты)

1. Создайте репозиторий на github.com, например `quran-zhattau`.
2. Загрузите в него **всё содержимое этой папки** (можно через кнопку «Add file → Upload files»
   или `git push`).
3. В репозитории: **Settings → Pages**.
4. **Source: Deploy from a branch**, ветка `main`, папка `/ (root)` → **Save**.
5. Через 1–2 минуты обновите страницу — появится адрес
   `https://ВАШ-ЛОГИН.github.io/quran-zhattau/`.

PWA и офлайн-режим работают только по **HTTPS** — у GitHub Pages он включён по умолчанию.
Файл `.nojekyll` уже добавлен, чтобы Pages отдавал все файлы как есть.

## Сборка APK через PWABuilder (после публикации)

1. Откройте https://www.pwabuilder.com
2. Вставьте адрес вашего сайта (из шага 5 выше) → **Start**.
3. **Package For Stores → Android**. Package ID укажите: `kz.quranzhattau.app`.
4. **Generate** → скачайте архив:
   - `app-release-signed.apk` — установить на телефон напрямую (sideload);
   - `.aab` + ключ — для публикации в Google Play.

Полная инструкция и альтернативный локальный путь (Capacitor + Android Studio) — в `BUILD-APK.md`.

## Через git (если удобнее командная строка)

```bash
cd quran-pwa
git init
git add .
git commit -m "Құран жаттау PWA"
git branch -M main
git remote add origin https://github.com/ВАШ-ЛОГИН/quran-zhattau.git
git push -u origin main
```

Затем включите Pages по инструкции выше.

## Технологии

Чистый HTML/CSS/JS без фреймворков. Текст Корана и переводы — `alquran.cloud`,
аудио — `cdn.islamic.network`. Прогресс хранится локально в браузере (`localStorage`).
Подключён строгий Content-Security-Policy, весь динамический контент экранируется.
