# Қуран Жаттау → установка как приложение (PWA)

## Что это даёт

Иконка на домашнем экране телефона, запуск в полноэкранном режиме без браузерной адресной строки, офлайн-доступ к уже открывавшимся сурам (текст кэшируется), быстрая загрузка.

Ограничение, о котором стоит знать: push-напоминания в фоне (при полностью закрытом приложении) требуют push-сервера — этого в комплекте нет. Напоминания работают, пока приложение открыто или свёрнуто.

## Установка за 5 минут (GitHub Pages, бесплатно)

1. Скопируйте файл `quran-jattau.html` в эту папку и переименуйте его в `index.html`. В папке должны лежать: `index.html`, `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`.
2. Зарегистрируйтесь на github.com (если нет аккаунта) и создайте новый публичный репозиторий, например `quran-app`.
3. Загрузите все 5 файлов: кнопка «uploading an existing file» → перетащите файлы → Commit.
4. В репозитории: Settings → Pages → Source: «Deploy from a branch» → Branch: `main`, папка `/ (root)` → Save.
5. Через 1–2 минуты приложение будет доступно по адресу `https://ВАШ_ЛОГИН.github.io/quran-app/`.

## Установка на телефон

Android (Chrome): откройте адрес → меню ⋮ → «Добавить на главный экран» / «Установить приложение».
iPhone (Safari): откройте адрес → кнопка «Поделиться» → «На экран "Домой"».

## Альтернативы GitHub Pages

Netlify Drop (drop.netlify.com) — просто перетащите папку в браузер, без регистрации git. Vercel, Cloudflare Pages — тоже бесплатны.

## Проверка, что PWA работает

Откройте сайт в Chrome → DevTools (F12) → вкладка Application → Service Workers: должен быть статус «activated». В адресной строке появится значок установки ⊕.

Прогресс хранится в браузере устройства. При переходе с локального файла на сайт прогресс не переносится автоматически — используйте «Скопировать прогресс» в настройках (вставить обратно пока нельзя — функция импорта в планах).
